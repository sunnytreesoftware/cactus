## Installation  
Install Laravel  
`
laravel new cactus
`  

Install Acorn  
`
composer require sunnytreesoftware/cactus
`  

Publish Assets  
`
php artisan vendor:publish --provider="Sunnytree\Cactus\CactusServiceProvider"
`

Go to your-website.com/cactus
